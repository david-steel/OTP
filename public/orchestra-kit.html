Token Economics

Cost = Compute Resources
Agent

Violin — Analysis

Precision work. Data analysis, pattern detection, measurement. The agent that reads the numbers.

Agent

Trumpet — Communication

Output and broadcasting. Email drafting, Slack messaging, client comms. The agent that speaks.

Agent

Drums — Execution

Action and throughput. Task completion, pipeline running, repetitive operations. The agent that does.

Agent

Piano — Coordination

Logic and orchestration. Scheduling, routing, decision trees. The agent that thinks across the whole system.

System

Sheet Music — OOS

The Organizational Operating System. Structured rules, claims, confidence levels. The shared intelligence document.

Coordination

Conductor — Orchestration

The coordination layer. Directs flow, resolves conflicts, ensures alignment. Not authority, coordination.

Flow

Flow Lines — Execution

Work moving through the system. Chaotic (overlapping, directionless) vs. structured (parallel, directional).

Scattered
Efficient
Cost

Tokens — Compute

Resource consumption. Scattered (wasted on duplication) vs. efficient (flowing through structured paths).

Usage
How to Use This Kit
Website & Blog
Use Before/After compositions in hero sections. Individual components as inline illustrations in blog posts. The 6-layer stack diagram in comparison articles.
Product & Onboarding
Musician icons as agent type selectors. Sheet music as OOS file previews. Conductor as the coordination dashboard icon. Tokens for usage/billing UI.
Pitch Decks & Marketing
Before/After as the core narrative slide. Individual musicians for "meet the agents" sections. Flow lines to show transformation stories.
Figma Import
Right-click any SVG on this page, copy the SVG code, and paste directly into Figma. All components use consistent 80x80 viewBoxes for easy alignment.
