# OTP AEO Question Map — Franchises & the Portfolio Feature

**Purpose:** editorial calendar for the franchise series. Target buyer: franchisors, multi-unit operators, franchise-development leaders, and PE-backed franchise platforms. The wedge: **a franchise IS a portfolio** (one franchisor, many locations), and OTP's portfolio feature rolls every location's hybrid human-and-agent chart up into shared super-metrics, with presets that let corporate set the standard once and lock it.

**Grounding:** `reference-portfolio-feature.md` (the real feature, with accuracy guardrails) + `research-franchise-orgs.md` (franchise-domain brief). Respect every guardrail: feature is in Labs/early access, no invented franchise customers, don't over-specify rollup math.
